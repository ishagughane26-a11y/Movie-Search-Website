# Movie Search Website

Movie Search Website is a simple and responsive web application built using HTML, CSS, and JavaScript. The website allows users to search movies dynamically and view movie details such as movie posters and release years.

---

## Features

* Search movies instantly
* Responsive user interface
* Attractive movie cards
* Smooth hover effects
* Dynamic movie filtering
* Beginner-friendly project

---

## Technologies Used

* HTML5
* CSS3
* JavaScript

---

## Project Structure

```bash
Movie-Search-Website/
│
└── index.html
```

---

## How to Run the Project

### Step 1

Download or clone the project.

### Step 2

Open the project folder in VS Code.

### Step 3

Run the project using:

* Live Server Extension
  OR
* Open `index.html` directly in browser

## Website Functionalities

* Displays movie cards with images
* Search movies using search bar
* Filters movies dynamically
* Responsive layout for mobile and desktop
